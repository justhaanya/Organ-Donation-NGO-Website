# Organ Donation NGO

## Live Demo
[View Live Demo](https://justhaanya.github.io/Organ-Donation-NGO-Website/)

## Project Overview
This project is a web application for an Organ Donation NGO. It aims to encourage people to pledge their organs and save lives. The website features a single-page design to navigate between home, about, impact, and donation sections, and includes a backend processing script for collecting donor details.

## Features
- **Single Page Application feel:** Smooth navigation between sections without page reloads using JavaScript.
- **Image Gallery:** Visuals showcasing the impact of different types of organ donations.
- **Donation Form:** A secure registration form for new donors, including an option to upload payment screenshots.
- **Admin Donor View:** Includes a backend interface to view registered donors (`donors_list.php`).

## Tech Stack
- **Frontend:** HTML5, CSS3, Vanilla JavaScript
- **Backend:** PHP
- **Database:** MySQL

## Database Setup
To run this project locally, you need a local server environment like XAMPP or WAMP.
1. Start Apache and MySQL in your XAMPP/WAMP control panel.
2. Create a MySQL database named `ngo_donation`.
3. Open `http://localhost/NGO/create_table.php` in your browser to automatically generate the required `donors` table.
4. You can now use the website at `http://localhost/NGO/index.html`.
